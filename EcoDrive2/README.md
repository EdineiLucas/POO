O Desafio:

1. Criar um método chamado gerarRelatorioViagens() dentro da classe Veiculo. Este método deve usar um laço de repetição (for) para percorrer a lista de viagens e exibir:

A data de cada viagem.
A distância de cada uma.
No final, a soma total de quilômetros percorridos (usando os dados da lista).

2. Criar um veículo e vincular um motorista categoria 'D'.
3. Realizar 3 viagens de distâncias diferentes (ex: 20km, 50km, 15km).
4. Chamar o método gerarRelatorioViagens().

O Desafio Extra: Tente realizar uma 4ª viagem com distância muito longa (que o combustível não suporte) e verifique se ela não aparece no relatório final.
